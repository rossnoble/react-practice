# Roomba Simulator Challenge

Build a single-screen application that simulates a Roomba vacuum cleaner moving around a 10x10 grid.

## Requirements
- [x] Render a 10x10 grid (100 cells total) with clear cell boundaries
- [x] Visual indicator showing Roomba's current position on the grid
- [x] Clear directional indicator (North ↑, East →, South ↓, West ←)
- [x] Clicking the "forward" button moves the Roomba forward one cell
- [x] Clicking the "turn right" button rotates the Roombaa 90° clockwise without moving
- [x] When Roomba tries to move beyond grid edge, stay in the same cell and turn right instead

## Bonus features
- [x] Auto-pilot mode (moves without buttons)
- [ ] Smooth movement transitions
- [ ] Keyboard controls (arrow keys instead of buttons)
- [ ] Click cell to move Roomba there
- [ ] Place obstacles inside grid
- [ ] Path tracking / visited cells
