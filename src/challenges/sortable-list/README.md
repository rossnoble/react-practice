# Sortable To-do List

Build a drag-and-drop sortable list of todo items using native drag events.

## Requirements
- [x] Render a list of todo list items
- [x] Each todo item will include a checkbox to toggle the status
- [x] Completed todo list items will strikethough the text
- [x] Provide visual feedback during drag
- [x] Items list is reordered by dragging into a new position

## Bonus features
- [x] Let draggable item move along X axis as well as Y axis
- [ ] Drop indicator showing where the item will be placed
