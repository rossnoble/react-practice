# Sortable To-do List

Build a drag-and-drop sortable list of todo items using native drag events.

## Requirements
- [ ] Render a list of todo list items
- [ ] Each todo item will include a checkbox to toggle the status
- [ ] Completed todo list items will strikethough the text
- [ ] Itmes can be reordered by dragging into a new position
- [ ] Update list state on drop

## Bonus feature
- [ ] Visual feedback during drag (e.g. placeholder, opacity change)
- [ ] Drop indicator showing where the item will be placed
